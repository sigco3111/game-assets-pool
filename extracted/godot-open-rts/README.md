# godot-open-rts — Metadata Only

> ⚠️ 이 게임의 자원은 **Godot 프로젝트에 임베드** (`source/match/units/*.gd` 형태) 되어 있어
> symlink 추출이 거의 의미가 없습니다. 그래서 본 폴더는 메타데이터만 보관합니다.

원본 저장소에서 유용한 자원:
- `media/` — 배너, 로고 (자체 디자인)
- `source/match/units/` — 각 유닛 프리팹 및 시각 정의
